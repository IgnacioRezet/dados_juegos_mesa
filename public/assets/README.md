# Assets de la mesa

Coloca aquí los recursos gráficos de El Anillo Único:

- **`map-comarca.jpg`** — el mapa de La Comarca que se usa como fondo de la mesa
  (`public/css/style.css` → `.map-bg`). Si el archivo no existe, se muestra un
  degradado pergamino como respaldo, sin romper nada.

  Recomendado: imagen horizontal, ~1920×1280 o mayor, optimizada (JPG/WebP).

Opcional (mejora futura): bordes decorativos por cultura (Hobbit/Elfo/Enano/Hombres)
exportados de los PDF oficiales, para los marcos de cada hoja. Por ahora el marco se
reproduce con CSS (acento de color por cultura).
